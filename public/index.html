How the Full Application Works: Upload an Excel file and specify a column name. The Django backend creates separate Excel files for each unique value in that column and automatically downloads them as a ZIP file.

File Upload

Drag & drop Excel files (.xlsx, .xls) with intuitive file validation and preview.

Smart Splitting

Split files based on any column values with automatic data processing and validation.

ZIP Download

Automatically download all split files packaged in a convenient ZIP archive.
